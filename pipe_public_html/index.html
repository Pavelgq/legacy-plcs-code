Программа "ТРУБА" предназначена для помощи в выборе полимерных труб ПРОТЕКТОРФЛЕКС, используемых при прокладке кабельных линий номинального напряжения до 500 кВ, выполненных однофазными или трехфазными кабелями.

Выбор труб осуществляется отдельно для каждого из двух вариантов прокладки: для случая размещения труб открытым способом в грунте на дне траншеи и для случая затягивания в грунт в канал, подготовленный при помощи технологии горизонтально-направленного бурения (ГНБ).
